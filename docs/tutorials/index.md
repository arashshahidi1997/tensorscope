# Tutorials

Learning-oriented walkthroughs that teach TensorScope by producing a concrete result.

Use this section when the reader is new to the workflow and needs a guided path, not a checklist.

In TensorScope, tutorials should usually:

- start from the demo dataset or a minimal local setup
- walk through one coherent workflow from start to finish
- explain what the reader should observe in the UI or API
- defer exhaustive details to reference pages

Planned tutorial topics:

- get the demo app running from `data/demo_lfp.nc`
- tour the linked timeseries, navigator, spatial, PSD, and spectrogram views
- follow one selection update through the backend and frontend
- implement a small new view or endpoint with guided steps
