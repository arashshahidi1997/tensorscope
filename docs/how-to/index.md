# How-To Guides

Task-oriented guides for concrete TensorScope tasks with a clear success condition.

Use this section when the reader already knows what they want to do and needs the shortest reliable path.

TensorScope how-to guides should be narrowly scoped and action-first. They should avoid architecture digressions unless a short rationale prevents mistakes.

Typical topics for this section:

- run the backend locally
- run the frontend locally
- launch the full dev UI
- generate or refresh demo data
- add a new view to the frontend registry
- add a new API route or DTO
- debug selection sync or slice-response failures
