TensorScope

Pure-Python core for the TensorScope migration.

Current scope:
- `tensorscope.core.state`: tensor registry plus validated selection state
- `tensorscope.core.schema`: canonical tensor schema normalization helpers
- `tensorscope.core.layout`: UI-agnostic layout preset descriptors
- `tensorscope.core.data`: modality abstractions and time alignment helpers
- `tensorscope.core.events`: event stream models and registry

This repo is in Phase 1 of the migration from the legacy Panel/HoloViews app in `cogpy`.
